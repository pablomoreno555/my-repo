scripts.turtlebot_controller.controller()[source]

This function initializes the ROS node and waits for the robot’s pose, controlling the robot publishing a cmd_vel, by relying on the rospy module.

The velocity is passed as a geometry_msgs::Twist message; you can find an updated version of this module in scripts.exercise2.
